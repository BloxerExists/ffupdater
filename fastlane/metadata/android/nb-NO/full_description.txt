Installerer og oppdaterer følgende nettlesere:

Nettlesere fra Mozilla eller basert på Firefox:
- Fennec F-Droid
- Firefox Browser, Firefox for Android Beta, Firefox Nightly
- Firefox Focus, Firefox Klar
- Iceraven
- Mull Browser
- Tor Browser, Tor Browser Alpha

Gode personvernnettlesere:
- Bromite, Bromite SystemWebView
- Mulch

Nettleser som er bedre enn Google Chrome:
- Brave Private Browser, Brave Browser (Beta), Brave Browser (Nightly)
- Krom
- Kiwi-nettleseren neste
- Vivaldi

Andre applikasjoner:
- Orbot

FFUpdater ser etter nye versjoner i bakgrunnen og laster dem ned. På Android 12+, eller enheter med rot-tilgang kan det også oppgradere programmer uten interaksjon fra brukeren.

Besøk https://notabug.org/Tobiwan/ffupdater/src/master/README.md eller bruk «Kildekode»-lenken nedenfor for mer info om sikkerhetsanalyser, andre distribusjonskanaler, begrensninger, feil, O-S-S, bygginstruks, brukte tredjepartsbibliotek, tekniske detaljer, osv.
