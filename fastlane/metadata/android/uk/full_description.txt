Встановлює та оновлює наступні браузери:

Браузери з Mozilla або на основі Firefox:
- Firefox браузер, Firefox для Android Beta, Firefox Nightly
- Firefox Focus, Firefox Klar
- Mull Browser

Добрі конфіденційні браузери:
- Bromite, Bromite SystemWebView

Браузери, що кращі, ніж Google Chrome:
- Brave Private Browser, Brave Browser (Beta), Brave Browser (Nightly)
- Kiwi Browser Next
- Vivaldi

FFUpdater перевіряє оновлення у фоновому режимі й завантажує їх. На Android 12+ або пристроях з root FFUPDater також може оновити програми без взаємодії з користувачем.

Відвідайте https://notabug.org/Tobiwan/ffupdater/src/master/README.md для отримання додаткової інформації.
