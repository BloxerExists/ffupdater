Instaluje i aktualizuje następujące przeglądarki:

Przeglądarki Mozilli lub oparte na Firefoksie:
- Fennec F-Droid
- Firefox Browser, Firefox for Android Beta, Firefox Nightly
- Firefox Focus, Firefox Focus Beta, Firefox Klar
- Iceraven
- Tor Browser, Tor Browser Alpha
- Thunderbird for Android, Thunderbird for Android Beta

Przeglądarki, które są lepsze niż Google Chrome:
- Brave Private Browser, Brave Browser (Beta), Brave Browser (Nightly)
- Chromium
- DuckDuckGo Browser
- Kiwi Browser Next
- Privacy Browser (F-Droid)
- Thorium
- Vivaldi

Inne zastosowania:
- FairEmail
- K-9 Mail / Thunderbird Mail
- Orbot

Program FFUpdater sprawdza w tle dostępność aktualizacji i pobiera je. Na urządzeniach z systemem Android 12+ lub rootem FFUpdater może również aktualizować aplikacje automatycznie.

Odwiedź: https://notabug.org/Tobiwan/ffupdater/src/master/README.md aby uzyskać więcej informacji.

Dzienniki zmian: https://notabug.org/Tobiwan/ffupdater/src/master/fastlane/metadata/android/pl/changelogs
