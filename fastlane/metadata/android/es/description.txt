Instala y actualiza los siguientes navegadores:

Navegadores de Mozilla o basados en Firefox:
- Fennec F-Droid
- Firefox Browser, Firefox for Android Beta, Firefox Nightly
- Firefox Focus, Firefox Focus Beta, Firefox Klar
- Iceraven
- Mull Browser
- Tor Browser, Tor Browser Alpha

Buenos navegadores para la privacidad:
- Bromite, Bromite SystemWebView
- Mulch, Mulch System WebView

Navegador que son mejores que Google Chrome:
- Brave Private Browser, Brave Browser (Beta), Brave Browser (Nightly)
- Chromium
- DuckDuckGo Browser
- Kiwi Browser Next
- Privacy Browser (F-Droid)
- Vivaldi

Otras aplicaciones:
- Orbot

FFUpdater comprueba las actualizaciones en segundo plano y también las descarga. En dispositivos Android 12+ o con root, FFUpdater también puede actualizar las aplicaciones sin la interacción del usuario.

Visita https://notabug.org/Tobiwan/ffupdater/src/master/README.md para más información.

Registros de cambios: https://notabug.org/Tobiwan/ffupdater/src/master/fastlane/metadata/android/en-US/changelogs
