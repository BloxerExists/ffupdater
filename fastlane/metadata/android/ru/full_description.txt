Скачивает и обновляет следующие браузеры:

Браузеры от Mozilla или основанные на Firefox:
- Fennec F-Droid
- Firefox Browser, Firefox для Android Бета, Firefox Nightly
- Firefox Focus, Firefox Focus Бета, Firefox Klar
- Iceraven
- Tor Browser, Tor Browser Альфа
- Thunderbird for Android, Thunderbird for Android Beta

Браузеры, которые лучше, чем Google Chrome:
- Brave Browser, Brave Browser Бета, Brave Browser (Nightly)
- Chromium
- DuckDuckGo Browser
- Kiwi Browser Next
- Privacy Browser (F-Droid)
- Thorium
- Vivaldi

Прочие приложения:
- FairEmail
- K-9 Mail / Thunderbird Mail
- Orbot

FFUpdater проверяет наличие обновлений в фоновом режиме и загружает их. На устройствах с Android 12+ или root FFUpdater может обновлять приложения без участия пользователя.

Посетите сайт https://notabug.org/Tobiwan/ffupdater/src/master/README.md или воспользуйтесь ссылкой "Исходный код" ниже для получения дополнительной информации о мерах безопасности, других каналах распространения, ограничениях, ошибках, FAQ, инструкциях по сборке, используемых сторонних библиотеках, технических деталях и т.д.

Переведенный журнал изменений: https://notabug.org/Tobiwan/ffupdater/src/master/fastlane/metadata/android/ru/changelogs
