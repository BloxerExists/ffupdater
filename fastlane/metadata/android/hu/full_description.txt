Telepíti és frissíti a következő böngészőket:

Mozilla vagy Firefox alapú böngészők:
- Fennec F-Droid
- Firefox Browser, Firefox for Android Beta, Firefox Nightly.
- Firefox Focus, Firefox Focus Beta, Firefox Klar
- Iceraven
- Mull Browser
- Tor Browser, Tor Browser Alpha

Jó adatvédelmi böngészők:
- Bromite, Bromite SystemWebView
- Mulch, Mulch System WebView

A Google Chrome-nál jobb böngészők:
- Brave Private Browser, Brave Browser (Beta), Brave Browser (Nightly)
- Chromium
- DuckDuckGo Browser
- Kiwi Browser Következő
- Privacy Browser (F-Droid)
- Vivaldi

Egyéb alkalmazások:
- Orbot

Az FFUpdater a háttérben ellenőrzi a frissítéseket, és le is tölti azokat. Android 12+ vagy root eszközökön az FFUpdater a felhasználó beavatkozása nélkül is képes frissíteni az alkalmazásokat.

További információkért látogasson el a https://notabug.org/Tobiwan/ffupdater/src/master/README.md weboldalra.

Fordított changelog: https://notabug.org/Tobiwan/ffupdater/src/master/fastlane/metadata/android/hu/changelogs
